# Repository Instructions

## Tasks Done by AI
- After completing any user-requested task, append a short dated summary to `tasks_done_by_ai.txt` in the project root.
- Do not overwrite existing entries in `tasks_done_by_ai.txt`; append only.
- Keep each summary brief and specific.

## Commands Run by AI
- After running any command, append the command name and the reason for running it to `commands_run_by_ai.txt`
- Do not overwrite existing entries in `commands_run_by_ai.txt`; append only.

## Architecture changes in AI
- If any code change results in architecture change, update `architecture.md` file.
